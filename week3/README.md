hw4-1 distance.py
所謂的大圓距離即為球面上兩點間的距離
假設(x1,y1)與(x2,y2)為兩點地理上的緯度與經度座標
兩點間的距離公式為:
d = radius*acos(sin(x1)*sin(x2)+cos(x1)*cos(x2)*cos(y1-y1))
使用者輸入以度數表示地球上的兩點的緯度與經度
接著顯示大圓距離
地球半徑約為6371.01km
公式內緯度與經度代表北與西 負數表示南與東

hw4-2 currency.py
將輸入的金額轉換為最小的貨幣單位
使用者輸入整數型態以表示總金額
接著輸出依序對50元、10元、5元、1元的組合
以使用最少硬幣來計算
